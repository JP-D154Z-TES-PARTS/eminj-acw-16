## 📋 概要

このリポジトリは、eminj-acwプロジェクトの仕様書バージョン管理と差分分析を行うためのリポジトリです。

## 📁 ディレクトリ構成

```
.
├── base/                    # 旧バージョン仕様書（eminj-acw-15-c-t）
│   ├── eminj-acw-15-c-t_spec/   # 仕様書ファイル
│   └── eminj-acw-15-c-t_src/    # ソースコードファイル
├── new/                     # 新バージョン仕様書（eminj-acw-16-a-t）
│   └── eminj-acw-16-a-t/        # 統合仕様書ディレクトリ
├── analyze_spec_diff.py     # 差分分析スクリプト
├── 仕様書差分分析サマリー.md  # 総合分析サマリー
├── 差分分析レポート.md        # 詳細差分レポート
├── 変更点一覧.md              # 変更点一覧
└── analysis_result.json     # 分析結果（JSON形式）
```

## 🔧 使用方法

### 差分分析の実行

```bash
python3 analyze_spec_diff.py
```

このスクリプトを実行すると、以下のファイルが生成されます：

1. **仕様書差分分析サマリー.md** - 差分分析の総合サマリー
2. **差分分析レポート.md** - 詳細な差分分析結果
3. **変更点一覧.md** - すべての変更点の網羅的なリスト
4. **analysis_result.json** - 機械可読形式の分析結果
